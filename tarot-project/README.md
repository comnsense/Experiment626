# Таро приложение

© 2026 comnsnense. Всички права запазени.

## Описание
Мобилно-оптимизирано уеб приложение за Таро с всички 78 карти, тъмна/светла тема и възможност за теглене на случайни карти.

## Функционалности
- Разглеждане на всички карти (Големи аркани 0-21, Малки аркани 1-14)
- Теглене на 1-5 случайни карти с обобщено тълкувание
- Тъмна и светла тема (запазва се в localStorage)
- Напълно адаптивен дизайн за мобилни устройства
- Социални връзки

## Технологии
- HTML5
- CSS3 (с CSS променливи за темите)
- JavaScript (ES6)
- Font Awesome икони
- Google Fonts (Nunito)
