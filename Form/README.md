# Experiment626 - HTML Forms Demo

A demonstration of HTML form methods (POST and GET) hosted on GitHub Pages.

## Live Demo
Visit: https://comnsense.github.io/Experiment626/

## Forms Included
- **POST Form**: Contact form with name, email, and message fields
- **GET Form**: Search form showing URL parameters in action

## Setup for POST Forms
Since GitHub Pages is static-only, this project uses [Formspree](https://formspree.io) for POST form handling:
1. Sign up at Formspree
2. Create a new form
3. Update the `action` URL in `Form.html`

## Local Development
1. Clone this repository
2. Open any HTML file in your browser
3. Edit and refresh to see changes

## Technologies
- HTML5
- CSS3
- JavaScript
- GitHub Pages
- Formspree (form backend)
